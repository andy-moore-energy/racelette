# raclette web app
a simple raclette eating application where the user clicks to add more raclette to their pile

## hosting
to be determined the best way to host this. target is to have it cheap or free

## programming language
whichever is best for the job

## platform compatibility
should work on mobile web browsers

## user experience
* the loading page shows a big title "Yum Yum" and then a button called "Chomp"
* theres an empty plate on the page, and a raclette grill
* when clicking the button, a raclette tray is added to a pile on the plate
* when the user is finished, they will click the button "I'm all done" and it will change page and show a congratulations image with a small celebration animation
