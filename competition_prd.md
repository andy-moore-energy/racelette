## Usage
* the game memory wipes at midnight UTC
* each day, there is a competition

### Opening page
* the user enters a name and picks an icon from a choice of different ascii winter sports characters
* the user then clicks "Chow down" to enter the next page

### Game page
* the game page is unchanged from the single player game
* in the backend, the app keeps track of the different users and their scores
* on the game page, at the top right, it shows who is winning currently, it shows their name and icon

### winners ceremony page
* on the ceremony page, the podium is populated by the top 3 characters
* the player can always opt to start again, which will wipe their score and take them back to the game page
